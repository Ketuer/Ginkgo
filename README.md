# Ginkgo (银杏)
点滴梦想编织成诗

## 🍁 简介
Ginkgo是基于nio编写的非阻塞通信框架，它包含客户端和服务端两部分，它实现了大部分的通信
功能，开发者只需关心需要传输的数据和服务端、客户端需要对数据如何处理，它完全去除了配置文件
而是采用注解开发。它同Dandelion的设计理念类似，都是为了简化开发流程。
